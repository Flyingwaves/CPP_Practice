### 练习 2.19
引用是对象的另一个名字，指针是指向另一种类型的复合类型

1.指针本身就是一个对象，在生命周期中可以修改指向不同对象。但引用不是对象，它与对象的初始值绑定在一起，并且不可更改

2.指针无须在定义时赋初值，也可以不初始化。但引用必须初始化。

3.指针可以赋值或拷贝，但引用无法重新绑定。
